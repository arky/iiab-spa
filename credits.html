# Content

Internet-in-a-Box includes a variety of educational content and applications from:

  • wikipedia.org
  • khanacademy.org
  • www.ck12.org
  • phet.colorado.edu
  • medlineplus.gov
  • hesperian.org
  • mawared.org
  • gutenberg.org
  • wiki.laptop.org
  • scratch.mit.edu
  • iicba.unesco.org
  • mathexpression.com
  • musictheory.net
  • healthphone.org
  • cdc.gov
  • wikem.org

Internet-in-a-Box also includes the work of content aggregators which we gratefully acknowledge:

  • RACHEL is a curation of selected offline content at oer2go.org.
  • Kiwix is a ZIM server and repository of Wikimedia and other content in a compressed ZIM file format at kiwix.org.
  • KA Lite is a server and repository of Khan Academy content in various languages at learningequality.org/ka-lite.

Internet-in-a-Box also contains a number of applications each of which has its own attribution information, which is included.
